# Kali Linux Part 1 -- Practice Exercises

These exercises help reinforce the basic Kali Linux and terminal
commands covered in **Part 1** of the notes.

Try to complete the exercises **without looking at the notes first**,
then verify your answers afterward.

------------------------------------------------------------------------

# Section 1 -- Basic System Information

1.  Open the terminal and display the **current username**.
2.  Display the **hostname** of your machine.
3.  Display the **current system date and time**.
4.  Display the **current working directory**.
5.  Show the **history of commands** you have executed.

------------------------------------------------------------------------

# Section 2 -- Navigating the File System

1.  Move to your **Desktop directory**.
2.  Confirm you are in the correct directory using a command.
3.  Move back to your **home directory**.
4.  List all files in the current directory.
5.  List **all files including hidden files**.

------------------------------------------------------------------------

# Section 3 -- Listing Files

1.  List files with **detailed information** such as permissions and
    file size.
2.  Display files with **human-readable file sizes**.
3.  Display the contents of the **Desktop directory without moving into
    it**.

------------------------------------------------------------------------

# Section 4 -- Creating Directories

1.  Create a directory named:

    test_folder

2.  Inside **test_folder**, create another directory called:

    notes

3.  Create the following structure using **one command**:

    projects/security

------------------------------------------------------------------------

# Section 5 -- Working With Nano

1.  Create a file named:

    practice.txt

2.  Inside the file write:

    I am learning Kali Linux.

3.  Save the file.

4.  Exit the nano editor.

5.  Open the file again using nano.

------------------------------------------------------------------------

# Section 6 -- Grep Practice

1.  Inside **practice.txt**, add the following lines:

    user Lloyd logged in\
    admin logged in\
    user attempted login\
    guest logged in

2.  Use **grep** to search for the word:

    user

3.  Display the **line numbers** where the word appears.

------------------------------------------------------------------------

# Section 7 -- Word Count

Using the file **practice.txt**:

1.  Count the **number of words** in the file.
2.  Count the **number of lines** in the file.

------------------------------------------------------------------------

# Challenge Exercise

Complete the following tasks:

1.  Create a directory called:

    cybersecurity

2.  Move into the directory.

3.  Create a file called:

    commands.txt

4.  Add at least **three Linux commands you learned** in this lesson.

5.  Save the file and exit.

6.  Use **wc** to count the number of lines in the file.

------------------------------------------------------------------------

# Goal

By the end of these exercises you should be comfortable with:

-   Navigating directories
-   Creating folders
-   Creating and editing files
-   Searching text using grep
-   Counting file contents using wc

Practice these commands multiple times until they feel natural.
